# FlutterPractice


#### 打开一个模拟器
open -a Simulator

#### 查看连接可用的设备和模拟器
flutter emulators

#### 运行指定的模拟器
flutter emulators —launch <emulator id>

#### 选择设备
cmd + shift + p -> flutter change device 选择设备

#### 调试
fn + f5

#### 更新
flutter upgrade

#### 更新包 
flutter packages get
flutter update-packages

#### 自动构建变动的文件
flutter packages pub run build_runner watch
